<h1>🎬 CineDream – Movie Discovery Web App</h1>

A production-ready movie discovery platform built with modern frontend technologies, integrating multiple public APIs to deliver real-time movie data and seamless user experience.

<h1>🚀 Live Demo<h1>
👉 https://cine-dream-palette.lovable.app

<h1>🎯 Project Overview<h1>
  
CineDream is a responsive web application that enables users to search, explore, and view detailed information about movies using external APIs. The project demonstrates strong fundamentals in frontend development, API integration, and deployment.

<h2>✨ Key Features<h2>
  
🔍 Real-time movie search functionality

🎞️ Browse trending and popular movies

📄 Detailed movie insights (ratings, release date, overview)

⚡ Fast loading with optimized API calls

📱 Fully responsive design

🔗 Multi-API integration (OMDb, TMDB, iTunes)

<h2>🛠️ Tech Stack<h2>

Frontend: React (Vite)

Styling: Tailwind CSS

APIs: OMDb API, TMDB API, iTunes Search API

Deployment: Lovable

Version Control: Git & GitHub

<h2>💡 Skills Demonstrated <h2>

API Integration & Data Handling

Asynchronous JavaScript (Fetch/Axios)

Component-based Architecture (React)

Environment Variable Management

Responsive UI/UX Design

Deployment & CI/CD Basics

<h2>📊 Impact / Highlights<h2>
  
Integrated multiple APIs into a single unified interface

Built a scalable and maintainable frontend architecture

Ensured secure handling of API keys using environment variables

Achieved fast load performance using Vite build optimization
